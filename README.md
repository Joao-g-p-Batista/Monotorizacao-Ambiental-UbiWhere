# Projeto-Monotoriza-o-ambiental-Ubiwhere
Projeto industrial de monotorização ambiental usando sensores low-cost e ESP32 para recolha de dados dos sensores e formatação para envio por MQTT para o servidor da empresa, onde será integrado na base de dados da empresa.
.
Projeto construido usando o framework de arduino para ESP32.
